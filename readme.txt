Add simple buttons to your product page that allow customers to quickly increase the quantity they wish to add to their cart

== Description ==

Allow your customers to quickly add multiple products to their charts while browsing your online store.

== Installation ==

Thanks for checking out Quantity Modifier! Set up is super simple and only takes a few minutes:

1. Upload the plugin files to the `/wp-content/plugins/quantityModifier` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Quantity Modifier screen to configure the plugin
